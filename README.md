# 🏠 家庭物品管理

一个功能完整的 **家庭物品有效期管理 PWA**（渐进式 Web 应用），支持 AI 物品识别、批量管理、家庭共享、云端同步。

> **在线体验**: [https://suyang37.github.io/home-inventory/](https://suyang37.github.io/home-inventory/)

---

## ✨ 功能特性

### 📱 核心功能
- **物品管理** — 添加、编辑、删除物品，记录名称、分类、位置、数量、生产日期、保质期
- **有效期追踪** — 自动计算过期日期，首页统计概览（正常/即将过期/已过期）
- **AI 拍照识别** — 调用摄像头拍照，使用 AI（Google Gemini / OpenAI / 通义千问）自动识别物品名称和分类
- **无有效期物品** — 衣物、工具等物品可标记为"无保质期"，不显示过期信息
- **批量管理** — 物品列表支持批量选择、批量删除
- **图片压缩** — 拍照后自动压缩图片（JPEG，最大宽度 800px），节省存储空间
- **搜索** — 首页搜索框，支持按名称、位置、备注搜索，保存搜索历史
- **位置管理** — 树形结构管理位置（如：厨房 → 冰箱 → 冷冻室），支持嵌套

### 👨‍👩‍👧‍👦 家庭共享
- **创建家庭** — 创建家庭并成为拥有者
- **邀请成员** — 生成邀请码，家人输入邀请码即可加入
- **角色管理** — 拥有者/管理员/成员/仅查看，四种权限级别
- **权限控制** — 管理员可管理成员角色，仅查看成员不能修改数据

### ☁️ 云端同步
- **坚果云 WebDAV** — 支持国内最稳定的 WebDAV 服务，免费版即可使用
- **自定义 WebDAV** — 支持任何 WebDAV 协议的云盘（NextCloud、天翼云盘等）
- **双向同步** — 先下载云端数据合并，再上传本地数据
- **数据安全** — 数据存储在用户自己的云盘，无需注册第三方服务

### 📊 数据管理
- **导出/导入** — JSON 格式备份和恢复数据
- **清除数据** — 一键清除所有本地数据

## 🚀 快速开始

### 在线使用
直接访问：[https://suyang37.github.io/home-inventory/](https://suyang37.github.io/home-inventory/)

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/suyang37/home-inventory.git

# 进入项目目录
cd home-inventory/pwa

# 使用任意 HTTP 服务器启动（如 Python）
python -m http.server 8080

# 或使用 Node.js
npx serve .
```

然后用浏览器打开 `http://localhost:8080` 即可。

> **注意**: 拍照功能需要 HTTPS 或 localhost 环境。

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| **原生 JavaScript (ES6+)** | 无框架，轻量级 SPA |
| **PWA (Service Worker)** | 离线缓存，可安装到主屏幕 |
| **localStorage** | 本地数据持久化 |
| **Google Gemini / OpenAI / 通义千问 API** | AI 物品识别（用户自备 API Key） |
| **WebDAV 协议** | 云端数据同步 |
| **CSS3** | 响应式设计，适配移动端 |
| **GitHub Pages** | 免费托管部署 |

## 📖 使用指南

### 添加物品
1. 首页点击右上角 **＋** 按钮
2. 点击 **📷 拍照识别（AI）** 拍照自动识别
3. 或手动填写名称、分类、位置等信息
4. 对于衣物、工具等无保质期物品，勾选 **无保质期**
5. 点击 **保存**

### 管理有效期
- 首页统计卡片显示 **全部/正常/即将过期/已过期** 数量
- 首页下方列出 **即将过期和已过期** 的物品
- 物品列表按过期状态排序（已过期 → 即将过期 → 正常）
- 无保质期物品不显示过期信息

### 批量管理
1. 打开 **物品** 标签页
2. 点击右上角 **📋** 按钮进入批量模式
3. 勾选要删除的物品
4. 点击 **删除选中** 批量删除

### 配置 AI 识别
1. 打开 **我的** → **🤖 AI 识别配置**
2. 选择 AI 服务商：Google Gemini（免费）/ OpenAI / 通义千问
3. 填入你的 API Key（仅存储在本地浏览器）
4. 点击 **测试连接** → **保存**
5. 拍照时将自动调用 AI 识别物品

### 家庭共享
1. 打开 **我的** → **家庭管理**
2. 创建家庭或输入邀请码加入（演示邀请码：`FAMILY`）
3. 拥有者可以管理成员角色、刷新邀请码

### 云端同步
1. 打开 **我的** → **☁️ 云端同步**
2. 选择 **坚果云** 或 **自定义 WebDAV**
3. 填写服务器地址、用户名、密码（坚果云请使用应用密码）
4. 点击 **测试连接** → **保存**
5. 点击 **立即同步** 上传/下载数据

## ⚠️ 注意事项

- 数据默认存储在浏览器 **localStorage** 中，清除浏览器缓存会导致数据丢失
- 建议定期 **导出数据备份** 或配置 **云端同步**
- 拍照功能需要设备支持摄像头，且需在 HTTPS 或 localhost 环境下使用
- AI 识别需要自行申请 API Key（Google Gemini 免费，OpenAI 按量付费，通义千问有免费额度）
- API Key 仅存储在本地浏览器中，不会上传到任何服务器
- 家庭共享功能使用模拟数据，数据仅存储在本地
